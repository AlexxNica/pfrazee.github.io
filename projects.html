Current Work

Local.js

A research project to create a new Web plugins architecture which is managed in the application-layer. Designed to give users control over the behaviors of Web applications by splitting the app into processes. Web Workers safely contain untrusted code, while the Page acts as a trusted kernel for executing privileged requests. All processes behave as services and communicate using HTTP.
project site repo

  • Intended purpose: user-extensible Web applications, browser "operating systems."
  • Novel technologies: HTTP interface to local functions and threads; on-demand delivery of Worker services (initial request creates); typed-link-driven behaviors.

STDRel.com

A library of link relation types (reltypes) which captures common Web API patterns. Part of a wider strategy to use browser-like techniques to drive host-independent Web applications. Made to support Local.js projects, but applies widely to the Web.
project site repo

  • Intended purpose: common Web standards.
