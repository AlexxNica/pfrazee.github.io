Current Work

Local.js

A research project to construct Web applications with multi-process, multi-origin components in a host environment. All processes behave as services and communicate using HTTP. Web Workers safely contain untrusted code, while the Page acts as a trusted kernel for executing privileged requests. project site repo

  • Intended purpose: user-extensible Web applications.
  • Novel technologies: uniform Ajax interface to local functions and components; on-demand delivery of Worker services (initial request creates); typed-link-driven behaviors.

STDRel.com

An open library of link relation types (reltypes) which captures common Web API patterns. Part of a wider strategy to use browser-like techniques to drive host-independent Web applications. Made to support Local.js projects, but applies widely to the Web. project site repo

  • Intended purpose: common Web standards.
